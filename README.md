# Rice Grain Classification Using Machine Learning

## Project Overview

This project focuses on classifying different varieties of rice grains using Machine Learning techniques. The dataset is preprocessed, analyzed, and used to train classification models that predict the rice grain variety based on its characteristics.

---

## Objectives

* Perform data preprocessing and cleaning.
* Analyze rice grain characteristics.
* Train and evaluate Machine Learning classification models.
* Compare model performance using evaluation metrics.

---

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Jupyter Notebook

---

## Dataset

This project uses the Rice Grain Classification dataset from Kaggle.

The dataset is not included in this repository. You can download it from Kaggle and place it in the project directory before running the notebook.

Dataset Source: Kaggle – Rice Classification Dataset


## Example features include:

* Area
* Perimeter
* Major Axis Length
* Minor Axis Length
* Eccentricity
* Convex Area
* Extent
* And other grain characteristics.
